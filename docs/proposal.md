# Project Title

Persistent Multi-Marker WebAR for Hijaiyah Learning

# Background

Current AR learning applications generally use one-to-one marker mapping. When a new marker is detected, the previous object disappears, limiting sequential learning experiences.

# Problem Statement

1. AR objects disappear when new markers are scanned.
2. Learners cannot build words progressively.
3. Existing AR systems rarely support persistent learning contexts.

# Objectives

1. Develop a persistent rendering mechanism.
2. Support multi-marker tracking.
3. Enable sequential Hijaiyah word construction.
4. Evaluate learning effectiveness.

# Target Users

- Early childhood learners
- Islamic elementary schools
- TPQ students

# Technology Stack

- Three.js
- MindAR
- TypeScript
- Vite

# Expected Outcomes

- Functional WebAR application
- Performance evaluation
- Usability evaluation
