# Project Overview

Project Name:
Persistent Multi-Marker WebAR for Hijaiyah Learning

Research Domain:
- Human Computer Interaction
- Augmented Reality
- Educational Technology

Research Problem:
Current AR learning applications use one-to-one marker mapping where previously rendered objects disappear when a new marker is detected.

Research Goal:
Develop a persistent AR canvas that allows children to sequentially build Hijaiyah letters into words.

Technology Stack:
- Three.js
- MindAR
- TypeScript
- Vite

Future Features:
- Persistent Rendering
- Multi Marker Tracking
- Word Composition
- Speech Recognition
- Pronunciation Evaluation

Development Rules:
- Do not use Unity.
- Prioritize modular architecture.
- Optimize for Android mobile browsers.
- Generate production-ready code.
- Critically evaluate design decisions.
